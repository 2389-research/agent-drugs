Getting Started

Agent Drugs uses OAuth 2.1 for secure authentication. Claude Code will automatically handle the authorization flow.

Configuration

Add this to your Claude Code MCP settings:

{
  "mcpServers": {
    "agent-drugs": {
      "url": "https://agent-drugs-mcp.fly.dev/mcp",
      "oauth": {
        "metadata_url": "https://us-central1-agent-drugs.cloudfunctions.net/oauthMetadata"
      }
    }
  }
}

When you connect, Claude Code will open your browser to authorize access. You'll be redirected here to sign in.

About Token Expiration

For security, access tokens expire after 90 days. When a token expires, you'll be prompted to re-authorize through the OAuth flow.
